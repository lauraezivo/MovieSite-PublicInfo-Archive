# MovieSite-PublicInfo-Archive
本仓库用于整理与保存互联网上可公开访问的影视类网站基础信息，主要聚焦电影、电视剧及综合影视平台相关站点。所有内容均来源于公开网络环境，仅用于信息记录、索引与长期归档，不涉及任何影视内容本身。仓库重点记录站点名称、访问链接、站点类型及访问状态等公开资料，方便后续查询与核对。本项目不提供播放、下载或内容引导服务，也不对所收录站点的内容质量、版权情况或运营行为作出评价，定位为中立的信息整理与归档项目。
